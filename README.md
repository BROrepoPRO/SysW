# SysW — Система автоматизации обработки заказ-нарядов авторемонта

**Назначение:** импорт, анализ и учёт данных заказ-нарядов из Excel в SQLite.

**Технический состав:**
- Модули VBA для Excel (импорт, парсинг, выгрузка)
- База данных SQLite (хранение заказ-нарядов, работ, запчастей)
- PowerShell-скрипты автоматизации
- Интеграция с Git и GitHub для контроля версий

**Команда проекта:**

| Роль | Обязанности |
|------|-------------|
| **Начальник мира** | Генерация идей, постановка задач, ключевые решения, архитектура и приоритеты. |
| **Главный помощник** | Координация, декомпозиция задач, промты для Code Assistant, контроль качества, интеграция результатов. |
| **Code Assistant (DevOps-команда)** | Написание кода, документации, тестов, автоматизация по готовым промтам. |

**Используемые технологии:** VBA, ADO, SQLite, Excel, Git, PowerShell, SourceCraft Code Assistant.
